# AmadeusBot
## 简介

 此项目是基于 Nonebot2 和 go-cqhttp 开发，以Postgresql及其他软件来实现插件功能的QQ群机器人
## 关于
机器人功能的实现学习和借鉴了许多大佬的项目，收获颇丰。（终于有机会实现让助手重现世间的目标）
